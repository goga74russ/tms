# 📋 Продуктовая стратегия TMS
> **Дата:** 4 марта 2026 | **Горизонт:** 6 месяцев (до сентября 2026)

---

## Позиционирование

**Для кого:** Транспортные компании 10-200 ТС (грузоперевозки РФ)
**Главный аргумент:** ЭПД-ready TMS с онбордингом за 30 минут вместо 6 месяцев 1С
**Ценовая ниша:** 15-50к₽/мес (между бесплатными таблицами и дорогими SAP/1С)

---

## 1. MUST — без этого не продать (законодательство)

### ЭПД (электронный путевой лист) — 🚨 дедлайн сентябрь 2026
Главный рыночный драйвер. Компании **уже ищут** решение. Конкуренты в основном не готовы.

| Компонент | Описание | Оценка |
|-----------|----------|--------|
| ГИС ЭПД | API Минтранса для обмена ЭПД | 2-3 нед |
| ЭТрН XML | Формат ФНС для электронной транспортной накладной | 1-2 нед |
| КЭП | Квалифицированная электронная подпись (CryptoPro CSP) | 1-2 нед |
| ЭДО | Интеграция с Диадок или СБИС (1 оператор) | 2 нед |

**Стоимость для клиента:** ~600-2000₽/мес (документы) + бесплатная КЭП от ФНС

### Платон (для грузовиков >12т)
~40% парков попадают. Без учёта в себестоимости — KPI неточные.

---

## 2. SHOULD — без этого не удержать клиента

### Мультитенантность
- `organizationId` на всех 22 таблицах + middleware-фильтрация
- Без этого нельзя запустить SaaS — только on-premise
- **Оценка:** 2-3 недели (миграция + middleware)

### Клиентский портал + трекинг
- Публичная страница отслеживания по номеру заявки (без логина)
- Портал: свои заявки, статусы, документы
- Push/SMS/email уведомления
- **Оценка:** 3-4 недели

### Оптимизация маршрутов
- "Объедини 5 заявок в один рейс с минимальным пробегом"
- OSRM (open-source, свой сервер) или Яндекс Маршрутизация API (15к/мес)
- Одна из топ-3 причин покупки TMS
- **Оценка:** 2-3 недели

### Топливные карты
- API Газпромнефть/Лукойл/Роснефть — закрытые, через партнёрку
- Реалистичный старт: CSV/Excel импорт чеков → потом API
- **Оценка:** 1-2 недели (импорт), 3-4 недели (API)

---

## 3. COULD — killer-фичи (чего нет у конкурентов)

### Умная аналитика
- Предупреждение об убыточном рейсе **до выезда**
- Аномалии топлива (факт vs норма → слив?)
- Прогноз просрочки документов (+14 дней)
- Рейтинг водителей (баллы за нарушения/эффективность)

### Быстрый онбординг
- Импорт из Excel: водители, машины, клиенты
- Шаблоны: "20 фур, Москва-регионы" → готовые настройки
- **Первый рейс через 30 минут после регистрации**

### Open API + Webhooks
- Публичный REST API с ключами
- Вебхуки на события (рейс создан, доставлен, etc.)
- **Оценка:** 1-2 недели

### White-label
- CSS vars + кастомный логотип для крупных клиентов (50+ машин)
- Требует мультитенантность
- +50% к чеку

## Конкурентные преимущества нашей TMS

1. **Архитектура лучше рынка** — monorepo, RBAC, event journal, offline-mobile
2. **ЭПД первые** — конкуренты не готовы к сентябрь 2026
3. **Онбординг 30 мин** — vs 3-6 месяцев 1С
4. **Тарификация 7 модификаторов** — ночь, выходные, срочность, возврат, простой
5. **152-ФЗ из коробки** — медданные изолированы, append-only журнал
