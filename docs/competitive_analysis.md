## 💡 Стратегия и ценообразование

**Таргет:** Mid-market (20-100 ТС) + Small (5-20 ТС)

| Тариф | Цена | Включено |
|-------|------|----------|
| **Free** | 0 ₽ | До 5 ТС, базовый функционал |
| **Pro** | 15,000 ₽/мес | До 50 ТС, трекинг, ЭПД, аналитика |
| **Business** | 35,000 ₽/мес | Безлимит, AI routing, API, приоритет |

**Наши козыри:**
1. Современный стек (конкуренты на PHP/1С) → быстрые фичи
2. API-first → лёгкая интеграция
3. ЭПД-ready → регуляторный барьер для конкурентов
4. Docker SaaS → внедрение за дни, не месяцы
5. Freemium → низкий порог входа
