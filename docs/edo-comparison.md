## Детальный анализ

### Контур.Диадок / Контур.Логистика

**Плюсы:**
- Лучшая на рынке API-документация (developer.kontur.ru)
- HTTP REST API, мультиязычные SDK
- «Контур.Логистика» — выделенный продукт именно для ЭПД
- Мобильное приложение с QR-кодами для водителей
- Поддержка всех типов ЭПД: ЭТрН, ЭПЛ, ЭЗЗ, ЭСВ
- Тестовая среда для интеграции
- Статус: лидер рынка ЭДО в РФ

**Минусы:**
- Стоимость чуть выше конкурентов при малых объёмах
- SDK нет нативного для Node.js/TypeScript (но HTTP API покрывает)

### СБИС (Тензор)

**Плюсы:**
- Широкая экосистема: ЭДО + бухгалтерия + CRM + HR
- Конкурентная цена
- REST API для интеграции
- Аккредитованный оператор ИС ЭПД
- Облачная ЭП (дистанционное подписание)

**Минусы:**
- API-документация менее структурирована, чем у Контура
- Нет отдельного SDK для TypeScript/JS
- Более сложная интеграция по сравнению с Диадоком

### 1С-ЭДО

**Плюсы:**
- Нативная интеграция с 1С:Предприятие
- Низкий порог входа если уже используется 1С

**Минусы:**
- Нет собственного API для внешней интеграции (работает через операторов)
- Нет мобильного приложения для водителей
- Ограниченная тестовая среда
- Не подходит для кастомной TMS без 1С

## 🏆 Рекомендация для TMS

### **Контур.Диадок + Контур.Логистика**

**Обоснование:**

1. **Лучший API** — REST HTTP, отличная документация, есть тестовая среда. Идеально для интеграции из нашего Node.js/Fastify стека
2. **Контур.Логистика** — специализированный продукт для ЭПД, не просто ЭДО
3. **Мобильное приложение** — водитель получает QR-код, ГИБДД сканирует. Интеграция с нашим driver app
4. **Все форматы ЭПД** — ЭТрН + ЭПЛ + ЭЗЗ + ЭСВ. Полное покрытие
5. **Масштабируемость** — 4 ₽/док при 5000+ документов

**Ориентировочный бюджет:**
- Подключение: 0 ₽
- 500 ЭПД/год: ~3 500 ₽
- КЭП для 3 сотрудников: ~12 000 ₽/год
- **Итого: ~15 500 ₽/год** (при малом объёме)

**План интеграции:**
1. Зарегистрироваться на developer.kontur.ru
2. Получить тестовый API-ключ
3. Реализовать отправку ЭТрН через REST API
4. Добавить кнопку «Отправить ЭПД» в путевой лист
5. Интегрировать QR-код в мобильное приложение водителя
