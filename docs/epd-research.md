# ГИС ЭПД — Исследование

> Дата: 04.03.2026 | Агент 8 | Sprint 5

## Что такое ГИС ЭПД

**ГИС ЭПД** (Государственная информационная система электронных перевозочных документов) — система Минтранса РФ для обмена электронными перевозочными документами между участниками грузо- и пассажироперевозок.

**Оператор:** Минтранс РФ.

## Сроки обязательного перехода

| Дата | Событие |
|------|---------|
| 01.03.2023 | Возможность оформления ЭПЛ (электронный путевой лист) в добровольном порядке |
| 01.09.2024 | Вступление в силу формата ЭТрН (приказ ФНС ЕД-7-26/383@) |
| **01.09.2026** | **Обязательный переход на ЭТрН/ЭПД для грузоперевозок** |
| 01.03.2027 | Конец переходного периода (бумажные ТН больше не принимаются) |

## Архитектура подключения

```
┌──────────┐     ┌──────────────┐     ┌──────────┐
│  TMS     │──▶──│ Оператор     │──▶──│ ГИС ЭПД  │
│ (наша    │ API │ ИС ЭПД       │     │ Минтранс │
```

**Прямого API к ГИС ЭПД нет.** Юрлица подключаются ТОЛЬКО через аккредитованных операторов ИС ЭПД.

## Формат ЭТрН XML

**Нормативный акт:** Приказ ФНС от 14.05.2024 № ЕД-7-26/383@ (вступил в силу 01.09.2024).

Электронная транспортная накладная состоит из нескольких титулов (файлов обмена), формируемых последовательно:

| Титул | Кто формирует | Содержание |
|-------|--------------|------------|
| **Титул 1** | Грузоотправитель | Данные отправителя, получателя, груз, маршрут |
| **Титул 2** | Перевозчик (при приёме груза) | Подтверждение приёма, ТС, водитель, состояние груза |
| **Титул 3** | Грузополучатель | Подтверждение получения, состояние при приёмке |
| **Титул 4** | Перевозчик (при доставке) | Статус доставки, пробег, показания одометра |

XSD-схемы размещены на [nalog.gov.ru](https://www.nalog.gov.ru). Каждый титул — отдельный XML-файл, подписывается КЭП соответствующего участника.

## Требования к КЭП

- Каждый участник (отправитель, перевозчик, получатель) подписывает свой титул **КЭП (квалифицированной электронной подписью)**
- КЭП руководителя организации выдаётся в **ФНС** (бесплатно)
- КЭП сотрудников — в аккредитованных УЦ (~3000–5000 ₽/год)
- Для сотрудников (не руководителей) требуется **МЧД** (машиночитаемая доверенность)
- Водители могут использовать **УНЭП ГК** (упрощённая неквалифицированная усиленная подпись через Госключ)

## Процесс подключения к ГИС ЭПД

1. **Выбрать оператора ИС ЭПД** (Контур.Логистика / СБИС / 1С-ЭДО)
2. **Заключить соглашение** с оператором
3. **Получить КЭП** для руководителя (ФНС) и сотрудников (УЦ)
4. **Оформить МЧД** для сотрудников, подписывающих документы
5. **Интегрировать TMS через API оператора** (REST/SOAP)
6. **Протестировать обмен** на тестовой среде оператора

## Стоимость

| Компонент | Стоимость |
|-----------|----------|
| Подключение к оператору | 0–5 000 ₽ (единоразово) |
| Пакет документов (год) | 3 500–20 000 ₽ (зависит от объёма) |
| КЭП руководителя (ФНС) | Бесплатно |
| КЭП сотрудника (УЦ) | 3 000–5 000 ₽/год |
| Токен (USB-ключ) | 1 500–3 000 ₽ |
| Интеграция через API | Включена в тариф оператора |

## Выводы для TMS

1. **Наша TMS формирует Титул 1 и Титул 4** (мы — перевозчик)
2. Передаём XML через API оператора → ГИС ЭПД
3. Нужна интеграция с одним оператором ИС ЭПД
4. Прототип ЭТрН генератора можно создать сейчас, подключение к оператору — ближе к сентябрю 2026
5. Для водителей — мобильное приложение с QR-кодом (уже есть кодовая база)
