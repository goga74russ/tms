## Правила языка

1. **Все ответы** пользователю — на **русском языке**.
2. **Документация и артефакты** (task.md, implementation_plan.md, walkthrough.md, ROADMAP.md, sprints/) — на **русском языке**.
3. **Весь код** — на **английском языке**: переменные, комментарии, docstrings, строки в коде, сообщения бота, UI-текст, API responses, логирование.
4. **Строки Telegram-бота** — на **английском языке** (пользователи бота видят английские сообщения).
5. **Описание пресетов** (preset.description) — на **английском языке**.
6. **Названия коммитов и PR** — на усмотрение пользователя, по умолчанию на английском.

> **ВАЖНО:** Код НИКОГДА не содержит русский текст. Только .md файлы документации могут быть на русском.
