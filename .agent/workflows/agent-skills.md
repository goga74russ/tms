# Скиллы для каждого агента TMS

Перед копированием промпта в новое окно — укажи агенту использовать эти скиллы.

## Агент 0 — Фундамент ✅ (завершён)
Скиллы: `typescript-expert`, `architecture-patterns`, `api-design-principles`, `docker-expert`

## Агент 2 — Осмотры + Путевые листы

## Агент 3 — Автопарк + Ремонты
Скиллы: `typescript-expert`, `frontend-developer`, `react-patterns`, `ui-ux-pro-max`, `testing-patterns`
> CRUD карточки, канбан ремонтов, справочники, DaData

## Агент 4 — Мобилка водителя
